📐 எப்படி கணக்கிடுது

⏳ Limit buy check — நான் சரியான விலையில வாங்குறேனா?

ஒரு limit buy price போட்டா, அது support-ல உட்கார்ந்திருக்கா (நல்ல entry) இல்ல resistance-ஐ துரத்துதா (மோசம்) என்பதை order-book + structure வைச்சு மதிப்பிடும். Risk : Reward = (entry→முதல் resistance) ÷ (entry→கீழ முதல் support). R:R ≥ 2 ஆனா நல்லது. ஒரு 0–100 buy score + Tamil verdict தரும்.

Sell targets (மேல எங்க விக்கலாம்)

• Breakeven = entry + ~0.1% fee.
• Ask walls = order book-ல மேலே இருக்கற பெரிய விற்பனை சுவர்கள் (median-ஐ விட ≥3× பெரிசு) — விலை இங்க தடைபடும்.
• SMA20 / SMA50 = 20/50-நாள் சராசரி; pump-கள் இங்க reject ஆகும்.
• +ATR = தினசரி சராசரி range வைச்சு projected target.

Support (கீழ எங்க தாங்கும் / risk)

• Bid walls = கீழே இருக்கற பெரிய வாங்கும் சுவர்கள்.
• 7d / 30d low = சமீபத்திய தளம்; உடைஞ்சா trend மோசம்.
• −ATR = கீழ்நோக்கி எதிர்பார்க்கும் range.
